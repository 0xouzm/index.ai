# 频道讨论：环球旅行

## 1. 频道定位

- **目标用户**：数字游民、深度旅行者、长期旅居者
- **内容调性**：实用、深度、避坑导向（非网红打卡）

---

## 2. 可用信源

### 2.1 政策类（官方）

| 来源 | 内容类型 | 更新频率 | 获取方式 |
|------|----------|----------|----------|
| 各国移民局官网 | 签证政策 | 政策变动时 | 爬虫 + 人工 |
| 外交部领事司 | 出行提醒 | 不定期 | RSS / 爬虫 |
| 各国大使馆公告 | 签证要求变更 | 不定期 | 爬虫 |
| Visa List / Passport Index | 签证互免信息 | 实时 | API |

### 2.2 数字游民资源

| 来源 | 内容类型 | 版权状态 |
|------|----------|----------|
| Nomad List（部分公开数据） | 城市评分/成本 | ⚠️ 需确认 |
| 各国数字游民签证官方页面 | 申请条件 | ✅ 官方公开 |
| Remote OK / We Work Remotely | 远程工作趋势 | ⚠️ 链接引用 |

### 2.3 实用信息

| 来源 | 内容类型 | 获取方式 |
|------|----------|----------|
| WikiVoyage | 旅行指南 | ✅ CC 协议，可全文 |
| 各国旅游局官网 | 目的地介绍 | ✅ 官方公开 |
| 航司/酒店官网政策 | 行李/取消政策 | 爬虫整理 |

### 2.4 社区 UGC（需授权）

| 来源 | 内容类型 | 风险等级 |
|------|----------|----------|
| 小红书/马蜂窝 | 游记攻略 | ❌ 高风险 |
| 个人博客（邀请投稿） | 深度游记 | ✅ 授权后可用 |

## 5. 周报功能

### 5.1 周报结构

```
🌍 环球旅行周报 - 2025年第X周

## 政策变动
- [泰国] 落地签延长至 60 天（生效日期）
- [日本] 新增数字游民签证细则

## 目的地推荐
- 本周精选：清迈雨季生存指南

## 签证到期提醒（个性化）
- 你的泰国签证将于 X 月 X 日到期

## 机票/住宿趋势
- 东南亚热门航线价格走势

---
By Index.ai | 查看完整档案 →
```

### 5.2 特色功能

- **政策预警**：检测到政策变动时，向订阅用户推送即时通知
- **个性化提醒**：用户可设置签证到期提醒（需登录）
